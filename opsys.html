Operating Systems Terminology

TERM DEFINITION
Operating System Software that controls the overall operation of a computer
Job Queue Where the jobs residing in mass storage wait for execution within a batch processing system
FIFO Storage organization in which objects are ordered in first-in, first-out FIFO fashion
Interactive Processing Operating systems that were developed to allow a program being executed to carry on a dialogue with the user through remote terminals
Time-Sharing Designed to provide service to multiple users at the same time
Multiprogramming Techique in which time is divided into intervals and then execution of each job is restricted to only one interval at a time
Application Software Consists of the programs for performing tasks particular to the machine's utilization
System Software Performs those tasks that are common to computer systems in general
Utility Software Consists of software units that extend or customize the capabilities of the operating system
Graphical User Interface (GUI) Objects that are to be manipulated, such as files and programs, are represented pictorially on the display icons
Shell Older user interfaces which communicated with users through textual messages using a keyboard and monitor screen
Kernel Internal part of an operating system which contains those software components that perform the very basic functions required by the computer installation
Device Driver The software units that communicate with the controllers to carry out operations on the peripheral devices attached to the machine
File Manager Coordinates the use of the machine's mass storage facilities
Memory Manager Coordinates the machine's use of main memory
Virtual Memory Large "fictional" memory space created by paging
Boot Strapping Often shortened to "booting," this procedure is performed by a computer each time it is turned on. This procedure transfers the operating system from mass storage into main memory
ROM Memory known as "read-only memory" because its contents can be read but not altered
Deadlock The condition in which two or more processes are blocked from progressing because each is waiting for the resource that is allocated to another
Spooling The technique of holding data for output at a later, but more convenient time
Super User (or Administrator) Establishes accounts and gains highly privileged access to the operating system by identifying him-or herself as the administrator during the login procedure
Multi-Factor Authentication Solution to secure computer accounts by requiring more than one piece of evidence to gain access
