Networking & The Internet

Network Classifications

  • LAN: Local Area Network
  • PAN: Personal Area Network
  • WAN: Metropolitan Area Network

Network Models

  • Bus Network: Machines are all connected to a common communication line
  • Star Network: One machine serves as a central focal point to which all the others are connected

Protocols

  • CSMA/CD Ethernet: (Carroer Sense, Multiple Access with Collison Detection) dictates that each message be broadcast to all the machines on the bus.
  • CSMA/CD Wifi: (Carrier Sense, Multiple Access with Collision Avoidance) the policy of trying to avoid collisions rather than trying to detect them.

Combining Networks:

  • Repeater: A device that passes signals back and forth between the two original buses
  • Bridge: Connects two buses, but it does not necessarily pass all the messages across the connection
  • Switch: A bridge with multiple connections, allowing it to connect several buses rather than just two
  • Router: Special purpose computers used for forwarding messages
  • Gateway: Serves as a passageway between the network and the outside world

Client/Server Model:

  • P2P: (peer-to-peer) whereas the client/server model involves one process (the server) providing a service to numerous other (clients)
  • Distributed System: Consist of software units that execute as processes on different computers
  • Cloud Computing: Huge pools of shared computers on the network can be allocated for use by clients as needed

Internet Addressing

  • Tier 1: consist of very high-speed, high-capacity, international WANs
  • Tier 2: More regional in scope and less potent in their capabilities
  • Tier 3: Operated by a single authority that is in the business of supplying Internet access to individual homes and businesses

IP Address:

  • ICANN (Internet Corporation for Assigned Names and Numbers): a nonprofit corporation established to coordinate the Internet's operation. www.icann.org
  • DNS: Internet-wide directory system
  • TLD (Top-Level Domain): Reflect the domain's classification
  • Country-code TLDs: Two-letter TLDs for each specific country. www.worldstandards

Internet Protocols:

  • FTP (File Transfer Protocol): Application for accessing another computer from a great distance
  • HTTP (Hyper Text Transfer Protocol): Network applications handled by webpages
  • SMTP (Simple Mail Transfer Protocol): A way that two computers on the network may interact when transmitting an email message from one host to another

World Wide Web:

  • URL (Uniform Resource Locator): unique address
  • Client-side activities: performed by a client
  • Server-side activities: performed by a server

Internet four-layer software:

  • Application Layer: where a message typically originates
  • Transport Layer: Used to send and receive messages over the Internet
  • Network Layer: In charge of maintaining the router's forwarding table and using that table to determine the direction in which to forward packets
  • Link Layer: In charge of receiving and transmitting the packets

Cybersecurity:

  • Malware: Many attacks incorporate the use of malicious software
  • Virus: Software that infects a computer by inserting itself into programs that already reside in the machine
  • Worm: An autonomous program that transfers itself through a network
  • Trojan Horse: A program that enters a computer system disguised as a desirable program
  • Spyware: Software that collects information about activities at the computer on which it resides and reports the information back to the instigator of the attack.
  • Phishing: A technique of obtaining information explicitly by simply asking for it
  • DDoS Attack: When the work of generating the attacks is distributed over many hosts

Protection:

  • Firewall: A primary prevention technique to filter traffic passing through a point in the network
  • Spam Filter: Firewalls designed to block unwanted email
  • Proxy Server: A software unit that acts as an intermediary between a client and a server with the goal of sheilding the client from adverse actions of the server
  • Antivirus Software: Used to detect and remove the presence of known viruses and other infections
  • Crytography: The study of sending and receiving messages securely in the presence of adversaries
  • Public Key Encryption: Techniques by which the encryption and decryption algorithms are known, and one of the keys is known, but the encoded message remains secure
